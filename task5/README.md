This project integrates sentiment analysis into a chatbot to detect
and respond appropriately to user emotions during interactions.

The chatbot identifies positive, negative, or neutral sentiment and
adjusts its responses to improve customer satisfaction.

## Features
- Real-time sentiment detection
- Emotion-aware chatbot responses
- Positive, negative, and neutral handling
- Streamlit-based user interface

## Technology Stack
- Python
- Streamlit
- Hugging Face Transformers

## How to Run
pip install -r requirements.txt  
streamlit run app.py

## Notes
A pre-trained sentiment analysis model is used to ensure accuracy and
efficient deployment without custom training.
